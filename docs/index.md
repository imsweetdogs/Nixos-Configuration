## ✨ Основные возможности

Ниже перечислено, что предоставляет эта конфигурация (полный список и подробности — в [README](../README.md)):

- ❄️ **Flakes** — версия-контролируемые зависимости и воспроизводимые сборки.
- 🏡 **Home-Manager** — декларативная настройка пользовательских окружений.
- 💽 **Disko** — описание разметки дисков в Nix.
- 🔥 **Sops-Nix** — безопасное хранение секретов.
- ⚠️ **Impermanence** — автоматическая очистка ненужных файлов.
- 📁 Чистая и расширяемая структура конфигурации, удобная для форков и PR.

## 🔧 Начало работы

1. Установите [Nix](https://nixos.org/) или загрузите live-образ NixOS.
2. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/pokedim13/NixOS-Configuration.git
   ```
