# system.root

Модуль задаёт базовые настройки для пользователя **root**.

---

## Опции

| Опция | Тип | По умолчанию | Описание |
|-------|-----|--------------|----------|
| `modules.system.root.enable` | `bool` | `false` | Включает модуль. |

---

## Что делает модуль

При `enable = true` выполняются два действия:

1. `users.mutableUsers = false` — запрещает изменять пользователей интерактивно (через `passwd`, `useradd` и т. д.); все изменения должны проходить через конфигурацию.
2. `users.users.root.hashedPassword = flake.conf.system.baseHashedPassword` — задаёт пароль *root* из заранее определённого хэша (см. `config/default.nix`).

Это гарантирует неизменяемость настроек root-пользователя и согласованность на всех машинах.

---

## Пример использования

```nix
modules.system.root = {
  enable = true;
};
```
