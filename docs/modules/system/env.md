# system.env

Модуль предназначен для объявления глобальных переменных окружения, доступных
в пользовательских сессиях.

---

## Опции

| Опция | Тип | По умолчанию | Описание |
|-------|-----|--------------|----------|
| `modules.system.env.enable` | `bool` | `false` | Включает модуль. |

Сейчас модуль предоставляет только опцию `enable`, но вы можете расширять её
дополнительными параметрами (например, добавлять собственные переменные) при
необходимости.

---

## Что делает модуль

При `enable = true` в секцию `environment.sessionVariables` записываются
следующие значения:

| Переменная | Значение | Назначение |
|------------|----------|------------|
| `XDG_BIN_HOME` | `$HOME/.local/bin` | Каталог для пользовательских скриптов и бинарей. |
| `PATH` | `[ "${XDG_BIN_HOME}" ]` | Добавляет `XDG_BIN_HOME` в PATH, сохраняя остальные элементы переменной. |

Вы можете добавить любые другие переменные, расширив модуль.

---

## Пример использования

```nix
modules.system.env = {
  enable = true;
};
```

После применения конфигурации новая переменная `XDG_BIN_HOME` станет доступна
во всех сессиях, а путь `$HOME/.local/bin` окажется в `$PATH`.
