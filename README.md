## ✨ Функциональность
- ❄️ Flakes - для точного управления зависимостями всей системы.
- 💽 Disko - для декларативного управления дисками.
- ⚠️ Preservation - Корень в оперативной памяти. Удаление лишних файлов по завершению сессии.

### Важно
* Некоторые хосты не имеют под собой disko для облегчения установки.
